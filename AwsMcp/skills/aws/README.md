# AWS MCP Tools Documentation

This directory contains documentation for all AWS MCP tools organized by service.

## Services

### CloudWatch
- **Logs Operations**: Filter, search, and manage CloudWatch logs
- **Metrics Operations**: Manage metrics and alarms
- **Insights**: Run complex queries on log data

### S3
- **Bucket Operations**: Create, delete, list buckets
- **Object Operations**: Upload, download, delete objects
- **Versioning**: Manage object versions

### ECS
- **Cluster Management**: List and describe clusters
- **Service Management**: Deploy and update services
- **Task Management**: Run and stop tasks

### ECR
- **Repository Management**: Create and manage repositories
- **Image Management**: List, describe, and delete images
- **Lifecycle Policies**: Manage repository lifecycle policies

### QuickSight
- **Dashboard Management**: List and describe dashboards
- **Analysis Management**: Work with analyses
- **User Management**: Manage QuickSight users

## Usage

Each tool references its specific documentation file when needed. The documentation provides:
- Parameter descriptions
- Return value formats
- Usage examples
- Important notes and warnings
