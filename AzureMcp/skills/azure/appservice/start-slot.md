# Start Deployment Slot

Start a deployment slot.

## Parameters
- **webAppName** (string): Web app name
- **slotName** (string): Deployment slot name
- **resourceGroupName** (string): Resource group name
- **subscriptionId** (string, optional): Azure subscription ID

## Returns
JSON object with success status and operation result.

## Example Response
```json
{
  "success": true,
  "message": "Slot staging started"
}
```
